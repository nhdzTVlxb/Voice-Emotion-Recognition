# 2025年睿抗智海算法调优 - 语音情绪识别解决方案

## 项目背景

作者参加2025年度睿抗机器人开发者大赛的智海算法调优赛道，本次赛题聚焦于语音情绪识别任务。

## 赛题特点与个人观察

### 赛制安排
- 比赛时间仅有三天，时间紧张（当然要考虑到数据量、轻量化模型等训练时间因素）
- 初始任务预告信息有限，方向不明确
- 作者提前准备了多个情绪识别方案，幸运地覆盖到了本次赛题（与编程赛制不同的是，这种赛道作为一个完整的项目，需要各方面的能力与经验，不局限于找模型、数据集出处等）

### 技术挑战
- 赛制限制下难以进行深度模型调优
- 官方未提供完整训练框架（我没有使用，具体有没有提供，我忘记了）
- 更侧重比赛经验和方法的快速应用
- 数据集规模较小，可能存在低样本学习的考察意图

### 数据来源
经作者确认，赛题数据来源于Kaggle竞赛数据集，但官方仅选取了部分样本。https://www.kaggle.com/code/mostafaabdlhamed/speech-emotion-recognition-97-25-accuracy

## 方案与成绩

### 核心成果
- **官方数据集基础方案**：F1分数达到 0.81
- **XGBoost优化方案**：F1分数提升至 0.95左右

### 技术遗憾
由于提交平台的技术问题，最优的XGBoost方案未能成功提交。

## 重要发现

### 数据扩充的影响
后续实验表明：
- 仅通过扩充数据集，基础模型（原F1 0.71-0.74）即可达到 0.9+ 的分数
- 证明数据量对模型性能有关键性影响

### 竞赛思考
- **选择大于努力**：作者觉得，在竞赛中还是在人生中，有时候还是选择大于努力的。
- **赛制是否要有限制**：部分选手通过数据扩充获得优势，影响了排名公正性，这个作者个人认为是有失算法调优这个目的的，（在0.71-0.74的old模型上就能达到0.9+的高分）。
- **个人坚持**：作者在比赛期间严格遵守规则，未使用外部数据扩充，虽然在江苏省排的一般般，但是在其他省能够是佼佼者；

## 解决方案开源

本项目开源作者在比赛期间开发的所有方案代码，供学术研究和技术交流使用。
